# 수입정산 매니저 MVP 개발 로드맵

영문 수입신고필증(PDF 스캔)과 한글 입고현황(xlsx)을 AI로 자동 매칭하고 부대비용을 품목별로 안분하여 이카운트 ERP 업로드 양식을 자동 완성한다.

## 개요

수입정산 매니저는 **수입 상품을 이카운트 ERP로 관리하는 중소 무역·유통사의 회계 담당자**를 위한 정산 자동화 도구로, 회계 담당자의 수작업 대조·계산·기입을 제거하기 위해 다음 기능을 제공합니다:

- **AI 자동 매칭**: 영문 신고필증과 한글 입고현황을 코드 매칭(1순위) + Claude 의미 매칭(2순위)으로 자동 연결
- **부대비용 자동 안분**: 운반비·수수료·기타를 품목별 수량 비중으로 안분하고 합계 검증
- **환율 자동 연동**: 입고일 기준 하나은행·서울외환 환율을 자동 조회하고 원화 환산
- **이카운트 양식 자동 생성**: 24컬럼 양식을 그대로 유지한 xlsx 파일 + 검증 리포트 다운로드

### 기술 스택

- **프레임워크**: Next.js 15 (App Router), React 19, TypeScript 5.6+ (strict)
- **스타일링/UI**: TailwindCSS v4, shadcn/ui (new-york), Lucide React
- **폼/검증**: React Hook Form 7.x, Zod
- **백엔드/DB**: Supabase (Auth, PostgreSQL, Storage, RLS), @supabase/ssr, @supabase/supabase-js
- **AI**: Claude API (Vision OCR + 영문↔한글 의미 매칭)
- **파일 처리**: pdf-lib / pdfjs-dist, xlsx (SheetJS), sharp
- **배포**: Vercel

## 개발 워크플로우

1. **작업 계획**
   - 기존 코드베이스를 학습하고 현재 상태를 파악
   - 새로운 작업을 포함하도록 `ROADMAP.md` 업데이트
   - 우선순위 작업은 마지막 완료된 작업 다음에 삽입

2. **작업 생성**
   - 고수준 명세서, 관련 파일, 관련 기능(F-ID), 의존성, 완료 기준, 구현 단계 포함
   - API/비즈니스 로직 작업 시 "## 테스트 체크리스트" 섹션 필수 포함 (Playwright MCP 테스트 시나리오 작성)
   - 새 작업은 빈 체크박스 상태로 작성하고, 완료 시 ✅로 갱신

3. **작업 구현**
   - 작업 파일의 명세서를 따름
   - API 연동 및 비즈니스 로직 구현 시 Playwright MCP로 테스트 수행 필수
   - 각 단계 후 작업 파일 내 단계 진행 상황 업데이트
   - 구현 완료 후 Playwright MCP를 사용한 E2E 테스트 실행, 테스트 통과 확인 후 다음 단계로 진행
   - 각 단계 완료 후 중단하고 추가 지시를 기다림
   - 작업 완료 체크: `npm run check-all` + `npm run build` 통과 확인

4. **로드맵 업데이트**
   - 로드맵에서 완료된 작업을 ✅로 표시
