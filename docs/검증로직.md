## 적용 대상 범위
- 4페이지부터 "ACE American Fire and Marine Insurance Company Korea, A Chubb Company" 문구가 나타나기 전까지의 모든 페이지는 수입신고필증으로 간주한다.
## 검증 조건
- 수입신고필증 내 각 항목의 35번(수량) 값의 총합계와 41번(수량) 값의 총합계는 반드시 일치해야 한다.
## 검증 로직
- 대상 페이지 범위 내 모든 수입신고필증 데이터 추출
- 각 문서의 35번 수량 값을 누적 합산 → total_qty_35
- 각 문서의 41번 수량 값을 누적 합산 → total_qty_41
- total_qty_35 == total_qty_41 여부 검증
## 예외 처리
- 두 합계가 일치하지 않을 경우 오류로 판단하며, 검증 실패 메시지를 반환한다.
## 오류 메시지 예시
- "수입신고필증의 35번 수량 합계와 41번 수량 합계가 일치하지 않습니다."
